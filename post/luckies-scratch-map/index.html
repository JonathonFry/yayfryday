Luckies Scratch Map

Jul 29, 2014
1 minute read

I bought this Luckies scratch map (from Amazon) a couple of weeks ago and I love it. I’ve only scratched off the countries that Jon and I have been to together. (I didn’t scratch Alaska/Hawaii as part of the USA though and decided to treat those seperately.) We need to go to more countries so I can scratch them off as it’s looking pretty unscratched!

I framed it in an IKEA Ribba frame (the one that has a 15 photo mount) which was almost perfect size - I just had to trim down some of the width to fit. Now I need to get it up on the wall and make some plans!
